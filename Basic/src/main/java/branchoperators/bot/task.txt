1. Нужно описать логику телеграмм бота которого купил школьник за 10 рублей у продавца с рынка.
за 10 рублей он умеет отвечать только 3 командам, если остальные ему неизвестны, то бот отрубается.

Запустите программу. После этого откроется консоль, где вы всегда видите ответы, теперь в неё можно вводить числа.
Ваша задача реализовать следующую логику.
Если ввести цифру 1, то программа ответит в консоль "Hello".
Если ввести цифру 2, то программа ответит в консоль "Buy".
Если ввести цифру 3, то программа ответит в консоль "What???".
Если ввести что-то иное, то программа ответит в консоль "I don't understand you, I'm turning off" и отключится.

Писать unit тесты на данный класс не нужно.

Подсказки:
1. Для сравнивания чисел нужно использовать знак двойного равенства "=="
2. Использовать нужно переменную number, именно в неё число поступает из консоли. Доп. переменные не нужны!
3. Чтобы программа отключилась нужно добавить следующий код:
bufferedReader.close();
break;
4. Принудительно завершить программу можно нажав на красный квадратик в InteliJ IDEA.
